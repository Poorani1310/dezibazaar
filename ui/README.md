# dezibazaar

This is an E-commerce website.

# Technologies Used

HTML, CSS JavaSCript, React - Vite, Mongodb, Express, Firebase

# Features provided in the ECommerce site

1. This site provides facilities to list the products based on the brand.
2. We can select products based on the categories like Mens fashion, womens fashion.
3. We can search the products by giving the in the search bar.
4. Can search products based on the brand or the product desc.
5. We can add items to the cart and we can checkout the items later.
6. Able to buy the product by clicking Buy now in the product.
7. Razor pay payment is enabled and can give upi, netbanking or card details to pay for the order.
8. We can also remove items from the cart.
9. Check out all the products in the Cart by summing the product prices.
10. Contact page has the enquiry form and we can send the enquiry as a mail to the site.
11. Google login is enabled usinf Firebase. 
12. Able to add to cart and buy products only when we login.
13. the site is reponsive to the window size.
